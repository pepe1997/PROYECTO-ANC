# Panel Operacional Integrado

Portal central para acceder a los modulos operacionales con una sola sesion.

## Ejecutar

```powershell
node server.js
```

Abrir `http://127.0.0.1:8080/`.

Usuarios iniciales: `admin` u `operador`. Contrasena inicial: `1234`.

Los proyectos originales ubicados junto a esta carpeta no son modificados.
